# Student Quick-Start Guide

> A 5-minute walkthrough for students. For the full guide, see [`user-guide.md`](./user-guide.md).

---

## 1. Login

1. Go to the platform URL (e.g. `https://exams.mewatitech.edu`).
2. Click **Login**.
3. Enter your email + password.

If you don't have an account yet, click **Sign up** on the landing page and pick the **Student** role.

---

## 2. Find your exam

After login, you land on **Dashboard**. Then either:
- Click an **Upcoming exam** card on the dashboard, **or**
- Click **My Exams** in the left sidebar.

You'll see exams with status **Live** (open now) or **Scheduled** (open later).

---

## 3. Start the exam

1. Click **Start exam** on a Live card.
2. Accept the **Fullscreen** permission when the browser asks.
3. The timer in the top-right starts counting down.
4. The **question palette** on the left shows all questions — blue = current, green = answered, gray = unanswered.

---

## 4. Answer questions

| Question type | What to do |
|---|---|
| Single MCQ | Click one radio button |
| Multi MCQ | Click every correct checkbox |
| True/False | Pick True or False |
| Fill blank | Type the answer in the input |
| Descriptive | Type a long answer in the text area |
| Coding | Write code in the editor; click **Run sample tests** to check |

Click **Save answer** any time. Answers are also auto-saved when you navigate to the next question.

---

## 5. Anti-cheating

When proctoring is on (most exams):
- **Don't** switch tabs or windows — each switch is logged.
- **Don't** exit fullscreen — exits are logged.
- **Don't** unplug or let your laptop sleep.
- The platform will **auto-submit** when the timer hits zero.

---

## 6. Submit

When you're done (or when time's up):
1. Click **Submit** in the top right.
2. You'll be redirected to a partial result page.
3. Objective answers are graded immediately. Descriptive / coding answers wait for your teacher.

---

## 7. View results

- Sidebar → **My Results**
- Click any row to see your full breakdown
- Click **Download PDF** to save your result certificate

> Once your teacher publishes your evaluation, you'll see their **comments** and final marks here, and you'll get an email + in-app notification.

---

## 8. Use the AI Code Assistant

Sidebar → **AI Assistant**

Try the four modes — SQL → Spark SQL, SQL → PySpark, Python → PySpark, and Explain code. Great for learning while practising!

---

## 9. Common things you'll need

| Want to | Where |
|---|---|
| Change profile name / phone | Avatar (top-right) → **Profile** |
| Change password | Avatar → **Settings** |
| See unread notifications | Bell icon (top-right) |
| See top performers | Sidebar → **Leaderboard** |
